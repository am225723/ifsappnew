# IFS Personalized Curriculum Framework

## Executive Summary
This framework enables dynamic curriculum personalization based on individual child wound assessment results, creating targeted healing pathways for each client.

---

## 1. Child Wound Types & Characteristics

### 1.1 Abandonment Wound ("The Lonely Child")
**Core Burden:** "I am unlovable unless I earn it"
**Protector Types:** Caretaker managers, People pleasers
**Behavioral Patterns:**
- Fear of being alone
- Clinging to relationships
- Hypervigilance to signs of rejection
- Difficulty with boundaries

**Curriculum Focus Areas:**
- Self-worth independent of others
- Healthy attachment patterns
- Setting boundaries
- Self-soothing techniques

---

### 1.2 Shame Wound ("The Unworthy Child")
**Core Burden:** "I am broken/fundamentally flawed"
**Protector Types:** Perfectionist managers, Inner critic
**Behavioral Patterns:**
- Constant self-criticism
- Difficulty accepting compliments
- Over-apologizing
- Hiding perceived flaws

**Curriculum Focus Areas:**
- Self-compassion practices
- Challenging inner critic
- Accepting imperfection
- Building self-acceptance

---

### 1.3 Neglect/Invisibility Wound ("The Lost Child")
**Core Burden:** "I don't matter"
**Protector Types:** Dissociative firefighters, Passive managers
**Behavioral Patterns:**
- Feeling invisible in groups
- Difficulty identifying needs
- Viewing self-care as selfish
- Emotional disconnection

**Curriculum Focus Areas:**
- Identifying and expressing needs
- Self-advocacy skills
- Reconnecting with emotions
- Building self-importance

---

### 1.4 Betrayal/Fear Wound ("The Terrified Child")
**Core Burden:** "The world is unsafe"
**Protector Types:** Controller managers, Hypervigilant parts
**Behavioral Patterns:**
- Difficulty trusting others
- Need for control
- Constant vigilance
- Fear of vulnerability

**Curriculum Focus Areas:**
- Building trust gradually
- Safety regulation techniques
- Vulnerability practices
- Releasing control patterns

## 2. Curriculum Personalization Algorithm

### 2.1 Assessment Score Interpretation
```
Score Range | Priority Level | Curriculum Intensity
0-8         | Low           | Awareness & Prevention
9-16        | Moderate      | Active Healing Focus
17-24       | High          | Primary Healing Target
```

### 2.2 Module Customization Rules

**Primary Wound (Highest Score):**
- 60% of curriculum content focused on this wound
- All core exercises adapted to this wound type
- Primary protector work centered here

**Secondary Wound (2nd Highest):**
- 30% of curriculum content
- Supporting exercises
- Integration with primary wound work

**Tertiary Wounds:**
- 10% awareness and prevention
- Brief acknowledgment in exercises

### 4.2 Activity Customization Matrix

#### Example: "Meeting Your Inner Child" Activity

**Original (Generic):**
```
Activity: Meeting Your Inner Child
- Close your eyes and imagine yourself as a child
- Notice what age appears
- Ask: "What do you need from me?"
- Listen with compassion
```

**Customized for Abandonment:**
```
Activity: Meeting Your Lonely Child
- Close your eyes and find the part that fears being alone
- Notice how young this part feels
- Ask: "What were you afraid would happen if you were alone?"
- Reassure: "I'm here now, and I'm not leaving"
- Practice: "You are lovable just as you are"
```

**Customized for Shame:**
```
Activity: Meeting Your Unworthy Child
- Close your eyes and find the part that feels flawed
- Notice what this part is trying to hide
- Ask: "What were you told was wrong with you?"
- Reassure: "There is nothing fundamentally wrong with you"
- Practice: "You are worthy of love and belonging"
```

**Customized for Neglect:**
```
Activity: Meeting Your Lost Child
- Close your eyes and find the part that feels invisible
- Notice what this part needed but didn't receive
- Ask: "What did you need someone to see in you?"
- Reassure: "I see you, and you matter"
- Practice: "Your needs are important and valid"
```

**Customized for Betrayal:**
```
Activity: Meeting Your Terrified Child
- Close your eyes and find the part that feels unsafe
- Notice what this part is protecting you from
- Ask: "What happened that made you feel unsafe?"
- Reassure: "You are safe now, and I will protect you"
- Practice: "It's safe to trust in small steps"
```

## 8. Technical Implementation

### 8.1 Database Schema (Supabase)

```sql
-- Clients table
CREATE TABLE clients (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  pin VARCHAR(6) UNIQUE NOT NULL,
  name VARCHAR(255),
  email VARCHAR(255),
  created_at TIMESTAMP DEFAULT NOW(),
  last_active TIMESTAMP
);

-- Assessment results table
CREATE TABLE assessment_results (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  client_id UUID REFERENCES clients(id),
  abandonment_score INTEGER,
  shame_score INTEGER,
  neglect_score INTEGER,
  betrayal_score INTEGER,
  primary_wound VARCHAR(50),
  secondary_wound VARCHAR(50),
  assessment_date TIMESTAMP DEFAULT NOW()
);

-- Personalized curriculum table
CREATE TABLE personalized_curriculum (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  client_id UUID REFERENCES clients(id),
  module_id VARCHAR(100),
  customized_content JSONB,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Progress tracking table
CREATE TABLE client_progress (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  client_id UUID REFERENCES clients(id),
  module_id VARCHAR(100),
  activity_id VARCHAR(100),
  completed BOOLEAN DEFAULT FALSE,
  completion_date TIMESTAMP,
  notes TEXT
);
```

### 8.2 Personalization Algorithm Pseudocode

```javascript
function generatePersonalizedCurriculum(assessmentResults) {
  // 1. Rank wounds by score
  const rankedWounds = rankWounds(assessmentResults);
  const primaryWound = rankedWounds[0];
  const secondaryWound = rankedWounds[1];
  
  // 2. Select module templates
  const moduleTemplates = getModuleTemplates(primaryWound);
  
  // 3. Customize each module
  const personalizedModules = moduleTemplates.map(module => {
    return {
      ...module,
      title: customizeTitle(module.title, primaryWound),
      content: customizeContent(module.content, primaryWound, secondaryWound),
      activities: customizeActivities(module.activities, primaryWound),
      exercises: customizeExercises(module.exercises, primaryWound)
    };
  });
  
  // 4. Add wound-specific resources
  const resources = getWoundSpecificResources(primaryWound);
  
  return {
    modules: personalizedModules,
    resources: resources,
    woundProfile: {
      primary: primaryWound,
      secondary: secondaryWound,
      focus: getPrimaryFocus(primaryWound)
    }
  };
}
```
