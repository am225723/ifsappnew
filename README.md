# The Luminous Self - IFS Program React App

A comprehensive web application for Internal Family Systems (IFS) therapy, designed to help users understand and heal their inner world through interactive tools, guided exercises, and educational resources.

## 🌟 Features

### Core Educational Content
- **IFS Cheat Sheet**: Quick reference guide to IFS principles, therapeutic techniques, and the 6 F's approach
- **10 Common Wounds**: Detailed exploration of inner child wounds with root causes and manifestations
- **Qualities of Self**: Interactive guide to the 8 C's and 5 P's of Self-energy
- **Resource Library**: Curated collection of books, videos, and audio resources with search functionality

### Interactive Tools
- **Parts Mapping**: Create and manage your personal parts map with detailed tracking of:
  - Part names and categories (Managers, Firefighters, Exiles)
  - Descriptions and triggers
  - Protective intentions
  - Full CRUD functionality (Create, Read, Update, Delete)

- **Guided Exercises**: Step-by-step meditations including:
  - Meet Your Parts (10 minutes)
  - Self-Energy Check-In (5 minutes)
  - Dialogue with a Part (15 minutes)
  - Progress tracking through exercise steps

- **Self-Assessment Tools**: Three comprehensive quizzes:
  - Identify Your Core Wounds
  - Identify Your Protective Parts
  - Self-Energy Assessment
  - Visual results with scoring and interpretation

- **Personal Journal**: Full-featured journaling system with:
  - Rich text entries with titles and dates
  - Mood tracking
  - Parts identification
  - Export/import functionality
  - Local storage persistence

## 🎨 Design Features

- **Beautiful UI**: Modern, gradient-based design with smooth animations
- **Responsive Layout**: Fully responsive design that works on desktop, tablet, and mobile
- **Intuitive Navigation**: Easy-to-use navigation with active state indicators
- **Color-Coded Categories**: Visual distinction between different parts and concepts
- **Interactive Elements**: Hover effects, transitions, and engaging user interactions

## 🛠️ Technology Stack

- **React 18**: Modern React with hooks
- **Vite**: Fast build tool and development server
- **React Router**: Client-side routing
- **Tailwind CSS**: Utility-first CSS framework
- **Lucide React**: Beautiful icon library
- **Local Storage**: Client-side data persistence

## 📦 Installation

1. Clone the repository
2. Navigate to the project directory:
   ```bash
   cd ifs-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser to `http://localhost:5173`

## 🚀 Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` directory.

## 🎯 Key Concepts Covered

### IFS Fundamentals
- The "No Bad Parts" philosophy
- Multi-mind vs. mono-mind model
- Self-energy and Self-leadership
- Parts categories and roles

### Therapeutic Process
- Parts mapping and identification
- The 6 F's approach (Find, Focus, Flesh Out, Feel Toward, Befriend, Fears)
- Unburdening process (5 steps)
- Self-compassion practices

### Healing Framework
- 8 C's of Self: Calmness, Curiosity, Clarity, Compassion, Confidence, Courage, Creativity, Connectedness
- 5 P's of Self: Patience, Persistence, Presence, Playfulness, Peace
- 10 Common Wounds: Abandonment, Neglect, Rejection, Criticism/Shame, Betrayal, Humiliation, Injustice, Loss/Grief, Emotional Invalidation, Trauma
